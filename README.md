# FX Assistant

本地优先的招行外汇牌价历史库 + 换汇账本 + 决策教练。

> **本仓库仅发布编译好的二进制，不含源代码。**

## 下载

前往 [Releases](https://github.com/harryrimPod/fx-assistant/releases) 下载 `fx-assistant.zip`，解压后运行 `fx-assistant.exe`（Windows）。

首次启动会自建空库，后续在你本机逐步积累历史。

## 这是什么

一个跑在你自己机器上的外汇工具：后台定时采集招商银行公开牌价、建立本地历史库、记录你的真实换汇成交、再用内置行为引擎做"提醒 / 教练"式决策辅助。**不接外部 LLM，不联网上传，数据永不出本机。**

## 核心特性

- **真·招行牌价**：每 10 分钟采集招行真实牌价入库；Hybrid 引擎真值优先，缺失时用市场中间价 × 溢价系数拟合兜底（界面以颜色区分真值 / 拟合）。
- **换汇账本**：手动录入真实成交汇率与金额，按币种统计持仓与成本。
- **决策教练**：从历史交易推导触发区间，给出买入 / 等待 / 忽略提醒；你的反馈回写本地，越用越懂你的习惯（≥5 条样本后激活标签与建议）。
- **纯本地引擎**：`reminder_engine` + `decision_learner` 全部本地运行，无云依赖、无第三方数据外发。
- **桌面壳**：内置 Electron 壳，也可直接用 Flask 后端 + 静态前端。

## 数据主权

你的交易历史、换汇策略、招行采集数据**只留本地、不上传、不汇集**。代码里物理上没有任何"上传"动作。每个下载者运行的是自己的采集、建自己的库、长自己的策略。
